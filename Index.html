Configuración de integraciones

Instrucciones de configuración

1Crear el Spreadsheet en Google Sheets

Ve a sheets.google.com y crea un nuevo spreadsheet en blanco. Copia el ID de la URL: https://docs.google.com/spreadsheets/d/ESTE_ES_EL_ID/edit

2Crear la carpeta en Google Drive

Ve a drive.google.com, crea una carpeta llamada "PODs Fotos". Ábrela y copia el ID del final de la URL: https://drive.google.com/drive/folders/ESTE_ES_EL_ID

3Crear el Apps Script

Ve a script.google.com → Nuevo proyecto → pega el código del archivo apps_script.js → reemplaza SPREADSHEET_ID y DRIVE_FOLDER_ID con los IDs del paso 1 y 2.

4Publicar el Apps Script

En el editor del script: Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Quién tiene acceso: Cualquier persona → Implementar. Copia la URL generada.

5Pegar la URL aquí arriba

Pega la URL del paso 4 en el campo "URL del Google Apps Script" de arriba y guarda.

6Publicar la app en GitHub Pages

Ve a github.com → Nuevo repositorio → sube el archivo pods_app_v3.html renombrado como index.html → Settings → Pages → Branch: main → Save. Tu URL será https://TU_USUARIO.github.io/TU_REPO
